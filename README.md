### Project Description
- A responsive weather dashboard that displays current weather conditions,
  1. 5-day forecasts
  2. 3-hour interval forecasts using the OpenWeather API.
 The project focuses on API integration, data transformation, and UI structuring for real-world frontend applications. DOM Manipulation codes

### Technologies Used
HTML5
CSS3
JavaScript (ES6+)
OpenWeather API
### Features
- City-based weather search
- Current weather display (temperature, humidity, wind, pressure)
- 5-day aggregated forecast
- 3-hour interval forecast
- Skeleton loading UI (ready state)
- Error handling for failed API requests
MapLibre integration hook (future enhancement ready)
### How to Run
1. Clone this repository
2. Open index.html in your browser OR
3. Replace API key in app.js
4. Enter a city name and search
### Lessons Learned
- Working with external REST APIs
- Transforming time-series weather data into meaningful UI components
- Handling asynchronous JavaScript using async/await and Promise.
### Challenges Faced
- Managing asynchronous API calls - efficiently
- Designing a layout that accommodates multiple forecast types in a single view
